## Requirements
- The app should have a view to display the list of all characters, all students, all staff. ‚úÖ
- The user should be able to select a character to go to a details page for the character. ‚úÖ
- The app should have a clear way to navigate between views. ‚úÖ
- A user should be able to favorite characters. ‚úÖ
- The user should be able to set their preferred house (Gryffindor, Slytherin, Hufflepuff, Ravenclaw). ‚úÖ
- The app should make use of colors and fonts that make aesthetic sense. ‚úÖ

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/GlHenrique/harry-potter-challenge
   cd harry-potter-challenge

   ```

2. Install dependencies:
   ```sh
   yarn
   ```

3. Run the development server:
   ```sh
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Testing
